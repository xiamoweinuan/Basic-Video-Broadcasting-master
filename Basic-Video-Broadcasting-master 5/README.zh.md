# 多人视频连麦直播

*[English](README.md) | 中文*

这个开源示例项目演示了如何快速集成 Agora 视频 SDK，实现多人视频连麦直播。

多人视频连麦直播的示例应用包含 Android，iOS，macOS，Web, 以及 Windows 平台。

多人视频连麦的示例应用展示了：

- 加入通话和离开通话；
- 主播和观众模式切换；
- 静音和解除静音；
- 切换摄像头；
- 选择分辨率、码率和帧率；
